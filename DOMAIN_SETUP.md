# 🌐 도메인 연결 가이드

## Vercel에 도메인 연결하기

### 1단계: Vercel 프로젝트에서 도메인 추가

1. **Vercel 대시보드 접속**
   - https://vercel.com 접속
   - 로그인 후 프로젝트 선택 (`unmyoung`)

2. **Settings 메뉴로 이동**
   - 프로젝트 페이지에서 **Settings** 클릭
   - 왼쪽 메뉴에서 **Domains** 클릭

3. **도메인 추가**
   - **Add Domain** 버튼 클릭
   - 구입한 도메인 입력 (예: `unmyoung.com` 또는 `www.unmyoung.com`)
   - **Add** 버튼 클릭

---

## 2단계: DNS 설정 (도메인 등록업체에서)

Vercel에서 도메인을 추가하면 **DNS 설정 방법**이 표시됩니다.

### 방법 A: 루트 도메인 연결 (예: `unmyoung.com`)

도메인 등록업체(가비아, 후이즈, 네임칩 등)의 DNS 관리 페이지에서:

#### 1. A 레코드 추가
```
타입: A
이름: @ (또는 비워두기)
값: 76.76.21.21
TTL: 3600 (또는 기본값)
```

#### 2. CNAME 레코드 추가 (선택사항 - www 서브도메인용)
```
타입: CNAME
이름: www
값: cname.vercel-dns.com
TTL: 3600
```

### 방법 B: 서브도메인 연결 (예: `www.unmyoung.com`)

#### CNAME 레코드 추가
```
타입: CNAME
이름: www (또는 원하는 서브도메인)
값: cname.vercel-dns.com
TTL: 3600
```

---

## 3단계: DNS 전파 대기

- DNS 변경사항이 전 세계에 전파되는데 **24-48시간** 소요될 수 있습니다
- 보통 **몇 분에서 몇 시간** 내에 적용됩니다
- Vercel 대시보드에서 도메인 상태를 확인할 수 있습니다:
  - ✅ **Valid Configuration**: 정상 연결됨
  - ⏳ **Pending**: DNS 전파 대기 중

---

## 4단계: SSL 인증서 자동 발급 확인

- Vercel은 **Let's Encrypt**를 통해 SSL 인증서를 자동으로 발급합니다
- 도메인 연결이 완료되면 자동으로 HTTPS가 활성화됩니다
- 보통 **몇 분 내**에 완료됩니다

---

## 주요 도메인 등록업체별 DNS 설정 위치

### 가비아 (Gabia)
1. 가비아 홈페이지 로그인
2. **나의 서비스 관리** → **도메인 관리**
3. 도메인 선택 → **DNS 관리** 클릭
4. 레코드 추가/수정

### 후이즈 (Whois)
1. 후이즈 홈페이지 로그인
2. **도메인 관리** → 도메인 선택
3. **DNS 관리** 또는 **네임서버 관리** 클릭
4. 레코드 추가/수정

### 네임칩 (Namecheap)
1. 네임칩 홈페이지 로그인
2. **Domain List** → 도메인 선택
3. **Advanced DNS** 탭 클릭
4. 레코드 추가/수정

### 구글 도메인 (Google Domains)
1. Google Domains 홈페이지 로그인
2. 도메인 선택
3. **DNS** 섹션 클릭
4. **Custom records**에서 레코드 추가/수정

## 환경 변수 업데이트

도메인 연결이 완료되면 **환경 변수**를 업데이트해야 합니다:

### Vercel 환경 변수 수정
1. Vercel 대시보드 → 프로젝트 → **Settings** → **Environment Variables**
2. `NEXT_PUBLIC_APP_URL` 찾기
3. 값 변경:
   - 기존: `https://unmyoung.vercel.app`
   - 변경: `https://unmyoung.com` (또는 연결한 도메인)
4. **Save** 클릭
5. **새 배포 필요**: 변경사항 적용을 위해 재배포하거나 자동 배포 대기

---

## 도메인 연결 확인 방법

### 1. Vercel 대시보드에서 확인
- **Settings** → **Domains**에서 상태 확인
- ✅ **Valid Configuration** 표시되면 연결 완료

### 2. 브라우저에서 확인
- 새 도메인으로 접속 시도
- 예: `https://unmyoung.com`
- SSL 인증서가 자동으로 적용되어 자물쇠 아이콘(🔒) 표시

### 3. DNS 전파 확인 도구 사용
- https://dnschecker.org 접속
- 도메인 입력 후 전 세계 DNS 서버에서 확인

## 문제 해결

### 도메인이 연결되지 않는 경우

1. **DNS 전파 대기**
   - 최대 48시간까지 기다려보세요
   - 보통 몇 시간 내에 해결됩니다

2. **DNS 레코드 확인**
   - 도메인 등록업체에서 레코드가 정확히 입력되었는지 확인
   - 오타나 공백이 없는지 확인

3. **TTL 값 확인**
   - TTL이 너무 높으면(예: 86400) 변경사항 반영이 느릴 수 있습니다
   - 3600 정도로 설정 권장

4. **Vercel 로그 확인**
   - Vercel 대시보드 → **Deployments**에서 배포 로그 확인
   - 오류 메시지가 있는지 확인

### SSL 인증서가 발급되지 않는 경우

1. **도메인 연결 완료 대기**
   - DNS 전파가 완료되어야 SSL 발급 가능

2. **Vercel 지원팀 문의**
   - 문제가 지속되면 Vercel 지원팀에 문의

## 추가 설정 (선택사항)

### www와 루트 도메인 모두 연결

1. **루트 도메인** (`unmyoung.com`) 연결
2. **www 서브도메인** (`www.unmyoung.com`) 연결
3. Vercel에서 자동으로 리다이렉트 설정 가능

### 커스텀 도메인으로 리다이렉트 설정

Vercel 대시보드 → **Settings** → **Domains**에서:
- 기본 도메인 선택
- 다른 도메인을 기본 도메인으로 리다이렉트 설정 가능
