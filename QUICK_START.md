# ⚡ 빠른 시작 가이드

## 🚀 가장 쉬운 방법 (추천!)

### 1단계: 서버 시작
**`start-server.bat`** 파일을 더블클릭하세요!

![서버 시작](https://via.placeholder.com/400x100/07080b/d4af37?text=start-server.bat+더블클릭)

### 2단계: 브라우저 열기
자동으로 열리지 않으면 브라우저 주소창에 입력:
```
http://localhost:3000
```

### 3단계: 즐기기! 🎉
- Cursor AI를 종료해도 서버는 계속 실행됩니다
- 코드를 수정하면 자동으로 새로고침됩니다

---

## 🛑 서버 중지

**`stop-server.bat`** 파일을 더블클릭하세요!

또는 서버 터미널 창에서 `Ctrl + C`

---

## 📱 모바일에서 접속하기

### 같은 Wi-Fi에 연결된 스마트폰/태블릿

1. **서버 시작 시 표시되는 Network 주소 확인**
   ```
   - Network: http://192.168.45.66:3000
   ```

2. **모바일 브라우저에서 해당 주소 접속**

3. **즐기기!** 📱✨

---

## ⚠️ 문제 해결

### "포트가 이미 사용 중" 오류
1. `stop-server.bat` 실행
2. 다시 `start-server.bat` 실행

### "페이지를 찾을 수 없음" 오류
1. 서버가 실행 중인지 확인 (터미널 창이 열려있는지)
2. 주소가 정확한지 확인: `http://localhost:3000`
3. 방화벽 확인

### 브라우저가 자동으로 열리지 않음
수동으로 브라우저를 열고 `http://localhost:3000` 입력

---

## 💡 팁

- **서버는 계속 켜두세요**: 개발하는 동안 서버를 계속 실행해두면 편합니다
- **자동 새로고침**: 코드를 저장하면 브라우저가 자동으로 업데이트됩니다
- **여러 탭 가능**: 여러 브라우저 탭에서 동시에 열어도 됩니다
- **Cursor와 독립적**: Cursor를 종료해도 서버는 계속 작동합니다
