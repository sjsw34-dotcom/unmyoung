# 🛒 고객 여정 및 구매 유도 분석

## 현재 페이지 구조

1. **Top Notice** - 한정 특가 안내
2. **Hero** - 메인 메시지 + CTA (상품 구성 보기)
3. **Hooking Section** - 후킹 메시지 + 상담 사례 + 리뷰 이미지
4. **Packages** - 상품 카드들 (선택하기 버튼)
5. **운세 이미지 섹션** - 배우자운, 재물운, 건강운, 직장운
6. **Mid Hooking Section** - "혹시 이런 생각 하고 계신가요?" + CTA (카카오톡)
7. **FAQ**
8. **CTA Section** - 최종 CTA (상품 다시 보기)
9. **Footer**

---

## ✅ 잘 되어 있는 부분

### 1. 명확한 구매 경로
- Hero → 상품 구성 보기 → 패키지 선택 → 결제 모달
- 패키지 카드에 "선택하기" 버튼이 명확함
- 결제 모달이 직관적임

### 2. 신뢰 구축 요소
- 상담자 이력 (Hero)
- 실제 상담 사례
- 리뷰 이미지
- FAQ

### 3. 긴급성/희소성
- "50명 한정", "10명 남음" (Top Notice)
- "10자리 남음" (Mid Hooking)

---

## ⚠️ 개선이 필요한 부분

### 1. 구매 흐름이 끊기는 지점

**문제점:**
- 패키지 섹션 → 운세 이미지 섹션 → Mid Hooking → FAQ
- 패키지를 본 후 구매 의도가 높은 시점에 다른 콘텐츠가 끼어듦
- Mid Hooking의 CTA가 카카오톡으로 가서 결제와 직접 연결 안 됨

**개선 제안:**
- 패키지 섹션 직후에 강력한 CTA 추가
- 또는 운세 이미지를 패키지 위로 이동

### 2. CTA 배치 문제

**현재 CTA 위치:**
1. Hero: "상품 구성 보기" ✅
2. Mid Hooking: "내 타이밍 확인하기" (카카오톡) ⚠️
3. CTA Section: "상품 다시 보기" ✅

**문제점:**
- Mid Hooking의 CTA가 카카오톡으로 가서 결제 흐름과 연결 안 됨
- 패키지 선택 후 바로 결제로 이어지는 CTA가 부족

**개선 제안:**
- Mid Hooking CTA를 "지금 바로 시작하기"로 변경하고 패키지 섹션으로 이동
- 또는 패키지 섹션 직후에 "지금 바로 시작하기" CTA 추가

### 3. 운세 이미지 섹션 위치

**현재 위치:** 패키지 섹션과 Mid Hooking 사이

**문제점:**
- 패키지를 본 후 구매 의도가 높은 시점에 다른 콘텐츠가 끼어듦
- 구매 결정을 방해할 수 있음

**개선 제안:**
- Hooking Section 내부로 이동 (상담 사례와 함께)
- 또는 패키지 섹션 위로 이동

### 4. 패키지 정보 부족

**현재:**
- 패키지명, 가격, 설명, 포함 내용만 표시

**개선 제안:**
- 각 패키지에 "누가 선택하나요?" 같은 타겟 설명 추가
- 가격 대비 가치 강조

## 🎯 권장 개선 사항

### 우선순위 1: 구매 흐름 개선

1. **패키지 섹션 직후 강력한 CTA 추가**
   - "지금 바로 시작하기" 버튼
   - 패키지 섹션으로 스크롤

2. **Mid Hooking CTA 수정**
   - 카카오톡 대신 패키지 섹션으로 이동
   - 또는 "상담 문의"와 "지금 구매하기" 두 개 버튼

### 우선순위 2: 콘텐츠 재배치

1. **운세 이미지 섹션 이동**
   - Hooking Section 내부로 이동
   - 또는 패키지 섹션 위로 이동

2. **패키지 섹션 강화**
   - 패키지 섹션 직후에 "지금 바로 시작하기" CTA 추가
   - 가격 대비 가치 강조

### 우선순위 3: 추가 개선

1. **패키지 비교표 추가** (선택사항)
2. **고객 후기 더 강조**
3. **보장/환불 정책 명시** (신뢰도 향상)
