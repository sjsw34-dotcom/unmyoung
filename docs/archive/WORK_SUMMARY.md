## 🐛 해결된 이슈

### 1. React Component 오류
- **원인**: `"use client"` 지시어 누락
- **해결**: 파일 상단에 추가

### 2. Viewport 경고
- **원인**: Next.js 16 새로운 방식
- **해결**: metadata에서 viewport export로 분리

### 3. Hero 섹션 파싱 오류
- **원인**: 중복/누락된 div 태그
- **해결**: 구조 재정립

### 4. 무한 로딩
- **원인**: 영상 파일 크기 (37MB)
- **해결**: 8MB로 압축

### 5. 반복되는 오류
- **원인**: Next.js 캐시
- **해결**: clean-cache.bat 생성
